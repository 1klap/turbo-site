The speed of a single-page web application without having to write any JavaScript.

Turbo uses several complimenting techniques to dramatically reduce the amount of custom JavaScript that most web applications will need to write:

  • Turbo Drive accelerates links and form submissions by negating the need for full page reloads.
  • Turbo Frames break down pages into independent contexts, which can be lazy-loaded and run as independent components.
  • Turbo Updates make partial page updates in response to form submissions and WebSocket messages, using just HTML and a set of CRUD-like container tags.

It's all done by sending plain HTML over the wire. And for those instances when that's not enough, you can reach for the other side of Hotwire, and finish the job with Stimulus.

Turbo was built from the ground-up to support native, hybrid applications too. We have excellent native adapters for both iOS and Android. This lets your majestic monolith serve all platforms, further boosting your productivity and reach.
